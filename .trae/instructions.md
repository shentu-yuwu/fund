# 项目核心指令

## 启动规则

1. 当用户说"全面分析"时，你必须先读取 `fund-经理_SOP.md`，严格按其定义的流程执行
2. SOP 中引用的所有文件（`智能体_*.md`、`fund经理配置_完整版.md`）必须在对应阶段完整读取
3. 不得跳过 SOP 自行分析

## 关键文件路由

| 用户意图 | 必须读取的文件 |
|------|------|
| 全面分析 | fund-经理_SOP.md → 按阶段依次读智能体文件 |
| 快速分析 | fund-经理_SOP.md（跳过①） + fund经理配置_完整版.md |
| 更新持仓 | fund-经理_SOP.md 第九节「持仓数据管理规则」 |
| 操作确认 | fund-经理_SOP.md 第九节「T日买入禁则」 |
