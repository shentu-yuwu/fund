# 持仓情况

> 使用方式：复制本文件为 `持仓情况.md`，填入你的真实持仓数据。此文件不会上传到 GitHub。

## 计算逻辑

```
更新日期  = 今天
净值日期  = T-1（昨结算，今早公布）
最新市值  = 持有份额 × 最新净值
成本总额  = 持有份额 × 成本净值
浮动盈亏  = 最新市值 - 成本总额
盈亏率    = 浮动盈亏 / 成本总额 × 100%
```
